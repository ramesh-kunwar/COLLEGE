# List

- List<T> class in c# represent a strongly typed list of object.
- Create, find, sort, search & manipulate list item.

```c#
List<int> numberList = new List<int x>;
numberList.Add(32);
numberList.Add(12);

List<string> another = new List<string>();
another.Add("Ram");
another.Add("Shyam");
```
